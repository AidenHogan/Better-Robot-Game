Summer project with some friends.

-The game takes inspiration from several sources:

-An interactive 'ecosystem' from Rainworld & Forever Winter

-Inventory, player control, combat, and some of the ecosystem interactions from Cogmind

-World map and interactions from Dwarf Fortress

-Automation aspect from Screeps: World


It will be a perma-death roguelike, with player control available.  However, the core gameloop will necessitate automation, with persistent code ensuring well built systems can get you back on track.  A low/no code option will be using the player controlled unit to take over others and hack them to steal their code and assimilate them into your faction.  Multiple factions will vie for power in this world, each with unique behavior (different code!) and traits.

The player will accumulate resources, create factories to make more powerful bots & items, and try to survive/defeat rival factions.  I'm not sure about story right now or what other goals or win conditions could be.
